# Synapse - Material Base para Registro

## Nome
Synapse

## Autor e proprietario
Daniel Sales

## Descricao geral
Synapse e uma plataforma SaaS empresarial com foco em unificar operacao, financeiro, pessoas, tecnologia e gestao em um unico sistema.

## Objetivo
Conectar setores, dados, pessoas, processos e decisoes em um unico ambiente.

## Tecnologias principais
- React
- Vite
- TypeScript
- Node.js
- Express
- tRPC
- Drizzle ORM
- Prisma
- PostgreSQL
- Azure App Service
- Vercel

## Modulos presentes no projeto
- base SaaS multiempresa
- empresas e grupos empresariais
- usuarios e permissoes
- financeiro
- frota e operacao
- RH
- TI
- tarefas
- notas
- Central do Daniel

## Diferenciais
- multiempresa com isolamento por empresa
- controle de acesso
- painel master
- base para auditoria e soft delete
- backend e frontend independentes
