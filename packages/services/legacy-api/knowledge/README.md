# Synapse Knowledge

Estrutura corporativa para RAG e base de conhecimento do Synapse.

Pastas:

- `docs/`: documentos homologados por fonte.
- `ingestion/`: manifestos e rotinas de ingestão.
- `metadata/`: metadados de origem, validade e aprovação.
- `index/`: artefatos de índice vetorial ou busca textual.
- `storage/`: área controlada para arquivos de conhecimento.

Regra do produto: a IA nunca deve inventar procedimento. Se uma fonte estiver pendente de ingestão ou sem homologação, a resposta deve escalar para atendimento humano.
